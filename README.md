💘 Ask Her Out – Fun Valentine Web Project

A playful and interactive mini web project designed to ask someone out in a creative (and slightly persistent 😏) way.

This project uses multiple pages, GIFs, and a fun "No button escape" trick to guide the user toward saying Yes.

🚀 Features
❤️ Cute animated GIFs using Tenor
😄 Multi-step interaction flow
🧠 Smart persuasion (multiple “No” pages)
🏃‍♂️ Moving "No" button (can’t click easily 😆)
🎯 Simple and lightweight (pure HTML, CSS, JS)
📂 Project Structure
📁 ask-her-out/
│── index.html        # First page (main proposal)
│── no1.html          # First "No" response
│── no2.html          # Second "No" response
│── no3.html          # Final "No" attempt page
│── yes.html          # Success page 🎉
│── style.css         # Styling file
│── script.js         # JS for moving "No" button
🛠️ How It Works
User lands on index.html
Clicks:
Yes → Redirects to yes.html 🎉
No → Moves through multiple persuasive pages
Final “No” button:
Starts moving randomly (JavaScript magic)
Makes it harder to reject 😄
📜 Technologies Used
HTML5
CSS3
JavaScript
Tenor GIF Embed API
▶️ How to Run

Just open the project locally:

1. Download or clone the repo
2. Open index.html in your browser

No server required. Simple and old-school — the way web started.
